About Celestial

Celestial is a server for managing the life cycle of systems across multiple hypervisors including AWS, Openstack, GCE, Digitalocean ...

It enables the provisioning of servers Using Puppet and deployment using Capistrano/Supernal all from the same UI.

Celestial controls multiple environments with fine grained user management.
